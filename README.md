# أكاديمية الصغار للناشئين 12–18

هذا المشروع يمثل الموقع الرسمي لأكاديمية الصغار، وهي منصة تهدف إلى تطوير مهارات الناشئين من عمر 12 إلى 18 سنة من خلال اختبارات الشخصية، برامج تدريبية، وبوتات كوتشينغ ذكية.

---

## 🎯 أهداف الأكاديمية
- مساعدة الناشئين على فهم شخصياتهم عبر اختبار MBTI.
- تعريفهم بالأنماط الـ16 للشخصية.
- تطوير مهارات التخطيط، التواصل، والذكاء العاطفي.
- تقديم باقات تدريبية متنوعة تناسب احتياجاتهم.
- توفير بوتات كوتشينغ تعمل كموجّه رقمي للناشئ.

---

## 📂 محتويات المشروع

### 1) الصفحة الرئيسية
`index.html`  
تحتوي على:
- قسم الترحيب  
- اختبار MBTI  
- الأنماط الـ16  
- الباقات  
- البوتات  
- التواصل  

---

### 2) التنسيقات
`styles/style.css`  
ملف يحتوي على جميع تنسيقات الموقع (الألوان، الخطوط، البطاقات، الأزرار).

---

### 3) البوتات الذكية
مجلد: `bots/`

يحتوي على:
- `planning-bot.html` — بوت التخطيط  
- `communication-bot.html` — بوت التواصل  
- `emotional-intelligence-bot.html` — بوت الذكاء العاطفي  

كل بوت يعمل ككوتش رقمي يجيب على أسئلة الناشئين ضمن تخصصه.

---

### 4) اختبار MBTI
مجلد: `mbti/`

يحتوي على:
- `test.html` — صفحة الاختبار  
- مجلد الأنماط: `types/`  
  - يحتوي على صفحات الأنماط الـ16 مثل:  
    - `type1.html`  
    - `type2.html`  
    - ...  
    - `type16.html`

---

### 5) الباقات
مجلد: `packages/`

يحتوي على:
- `package1.html` — الباقة الأولى  
- `package2.html` — الباقة الثانية  
- `package3.html` — الباقة الثالثة  

---

## 🌐 طريقة تشغيل الموقع

بعد رفع الملفات إلى GitHub:

1. افتح إعدادات المستودع (Settings).
2. انتقل إلى قسم Pages.
3. اختر:
   - Branch: `main`
   - Folder: `/root`
4. احفظ الإعدادات.
5. سيظهر رابط موقعك على GitHub Pages.
